# Lights-Out-Java
The game of Light's out with changeable dimensions and replayability

## How to play:

1.  Download files
2.  Extract zip file (If using download zip option)
3.  Navagate to LightsOutJava folder
4.  Click runLightsOutJava.bat

Enjoy!!!
